## 📖 Project Overview

This project is a fully interactive **Olympics Analytics Dashboard** built using **Python, Pandas, Streamlit, and Plotly**.

The dashboard enables users to explore Olympic Games data spanning more than a century and uncover insights about:

* Medal-winning nations
* Athlete performance
* Olympic participation trends
* Country-wise dominance
* Gender participation
* Historical Olympic statistics

The project transforms raw Olympic datasets into an intuitive analytics platform with dynamic filters and interactive visualizations.

## 🚀 Run Locally

Clone the repository:

```bash
git clone https://github.com/SAKETHRAM22/OLYMPICS-DATA-ANALYSIS.git
```

Move into the project folder:

```bash
cd OLYMPICS-DATA-ANALYSIS
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
